# Mini Course Subscription App

This is a **Mini Course Subscription App** built with **React (frontend)** and **Node.js + Express (backend)**, connected to **MongoDB**. Users can view courses, sign up, log in, and manage their subscriptions.

---

## Features

- User Authentication (Signup & Login)  
- Browse available courses  
- Subscribe to courses  
- View subscribed courses  
- Protected routes for logged-in users  
- Responsive UI with Tailwind CSS  

---

## Tech Stack

- **Frontend:** React, React Router, Axios, Tailwind CSS  
- **Backend:** Node.js, Express  
- **Database:** MongoDB  
- **Authentication:** JWT (JSON Web Tokens)
